Judul Aplikasi Web: Tell Tale

Deskripsi:
Tell Tale atau yang jika diartikan secara bahasa adalah ‘menceritakan kisah’. Tell Tale sendiri suatu web yang diperuntukkan khusus bagi para penulis fiksi di luar sana.
Web ini menggabungkan fungsi aplikasi Wattpad dan Story Plotter yang dapat ditemukan di play store.

Tell Tale sendiri dibuat untuk membantu penulis (baik pemula maupun pro) dalam membangun kisah yang ingin mereka ceritakan pada orang lain. Secara kasar, Tell Tale adalah suatu platform menulis namun dengan rincian tambahan yang dapat membantu para penulis tetap fokus pada alur cerita yang telah mereka rancang.

Library:
Frontend: React JS
Backend: Python Pyramid
Database: PostgreSQL

Fitur Utama:
- Publikasi/draft cerita
- Penyimpanan Karakter
- Penyimpanan timeline (penanggalan)

Referensi: Aplikasi Wattpad dan Story Plotter dari play store
